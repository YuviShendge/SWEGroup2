## ** 1. Clone this Repository**
Open a terminal:
git clone https://github.com/YuviShendge/SWEGroup2.git
cd SWEGroup2

## ** 2. Set Up Backend**
cd backend
npm install
npm start

## ** 3. Set Up Frontend**
cd ../frontend
npm install
npm start
